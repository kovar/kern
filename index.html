How to Use

1. Connect to your scale

USB (Chrome/Edge only)
Click USB in the toolbar and select your serial port. This uses the Web Serial API for direct hardware access — no extra software needed.
Bridge (any browser)
Start the bridge in a terminal: uv run bridge.py, then click Bridge. This relays data over WebSocket so Firefox, Safari, and other browsers can talk to the scale.

2. Toolbar buttons

Tare
Zero out the current weight on the scale (subtracts container weight).
Zero
Set the scale's zero reference point.
SI
Request a single immediate weight reading from the scale.
Play / Stop
Start or stop automatic polling. The scale is queried at the interval shown in the ms field (default 1000 ms).
Record / Save
Record begins capturing readings. Save stops recording and downloads a CSV file with timestamps and values.

3. Chart controls

Window sets how much history to show (30 s to 30 min). Y Min/Max lock the vertical axis. Reset returns to auto-scaling. Clear erases all data and resets statistics.

4. Demo mode

Click Demo to generate fake scale data and try out the app without hardware.

5. Typical workflow

Connect → Click Record → Click Play → watch live readings and chart → Click Stop → Click Save to download CSV.

Running locally

Don't open index.html directly — ES modules need a web server. Run uv run serve.py and open http://localhost:8000.
